Videoüberwachung 2.0

Project Description

Videoüberwachung 2.0 is a surveillance software that aims to provide a user-friendly and aesthetically pleasing solution for monitoring a variety of cameras from different manufacturers. It was developed during my time at Spreewunder for Adeck Systems GmbH.

My contribution

As the frontend developer on the Videoüberwachung 2.0 project, my role involved working with other developers and the design team to create the software.

This included following the design guide to accomplish the overall look and feel of the software, as well as implementing the user experience. I also developed interactive features and visualizations to enhance the functionality of the software. I worked closely with the backend developers to ensure that the frontend interface was properly integrated with the underlying system. My duties also included testing the software to ensure that it was user-friendly and performed well, as well as troubleshooting and debugging any issues that arose.

Technologies

Throughout this project, I utilized and worked with several technologies including, but not limited to:

  • Angular
  • Zeplin
  • MQTT
  • ONVIF
